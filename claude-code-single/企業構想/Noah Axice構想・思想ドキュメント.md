# 1. 概要

**Noah Axice** は  
人・企業・政府・技術・文化をつなぎ、  
プロジェクトを設計・統括する企業である。

特定の業界の会社ではなく、  
社会の中で様々な領域を横断して  
プロジェクトを生み出す **ネットワーク型企業** を目指す。
