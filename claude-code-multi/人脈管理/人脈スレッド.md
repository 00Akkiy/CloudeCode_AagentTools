# 人脈スレッド

会長（あつき）が出会った人・繋がった人を**チャット感覚でそのまま投稿**するスレッド。

キャスティング部が投稿内容を読み取り、`人脈台帳.csv` に自動登録する。

---

## 使い方

> 「〇〇さんと繋がった。ダンサーで大阪在住。インスタ5万人いる。」

これだけ書けばOK。詳細は後から補足できる。

---

## 投稿テンプレ（コピペして使う）

```
名前：
連絡先：（Instagram / LINE / X など）
どこで会った：
何ができる人：
俺との関係値：
稼働状況：（フリー / 副業可 / 多忙 / 不明）
補足：
```

> **最低限「名前」と「どこで会った」だけでOK。残りは補足で。**
