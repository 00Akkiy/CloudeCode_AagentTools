## ■ 判断：修理しない → 直接125cc移行（2026-04-08最終更新）

**現状渡し売却額：12,000円（確定）**

### ガソリン代の差が決定的
```
月間走行距離前提：91h × 20km/h ≈ 1,800km/月

ハイゼット（10km/L・160円）：1,800 ÷ 10 × 160 = 28,800円/月
PCX125（実燃費45km/L・160円）：1,800 ÷ 45 × 160 =  6,400円/月

ガソリン差額：22,400円/月
```

### 月間コスト全比較（ローン終了後）
| | 車 | PCX125レンタル |
|---|---|---|
| 保険+駐車場 | 18,000円 | 0円（レンタル込み） |
| レンタル | 0円 | 20,000円 |
| ガソリン | **28,800円** | **6,400円** |
| **合計** | **46,800円** | **26,400円** |

**毎月20,400円バイクの方が安い。**

### コストコ配達の優位性を考慮しても
```
コストコプレミアム：2,500円/h（4,000 - 通常1,500）
バイクのコスト優位：20,400円/月

損益分岐：20,400 ÷ 2,500 = 月8時間以上のコストコ必要
→ 週2時間以上コストコが取れないなら、バイクが得
→ コストコ以外の案件は車とバイクで単価差なし（確認済み）
→ 雨の日単価も車バイクで差なし（確認済み）
```

### 「5万修理して壊れたら」のリスク試算
```
修理→また壊れた場合の総コスト：
  修理費：    50,000円（消える）
  売却損：    46,000円（12,000 - 58,000）
  免許+ギア：  95,000円
  合計：     191,000円

最初からバイク移行した場合：
  売却損：    46,000円
  免許+ギア：  95,000円
  合計：     141,000円

差額：50,000円の損失リスク
```

**免許は一生使える資産。修理費は消えるだけ。**

### 移行コスト・回収期間
```
初期費用合計：
  売却損：    46,000円
  普通二輪免許：70,000円
```

## ■ 疲労比較：原付 vs 125cc

現在、原付でフードデリバリーをお試し中。「めちゃくちゃ疲れた」の原因と改善見込み。

**原付が疲れる主な理由：**
| 原因 | 内容 |
|---|---|
| 30km/h制限のストレス | 後続車に常に追われる精神的消耗 |
| エンジン高回転 | 30km/h維持でもエンジンが唸る → 振動が蓄積 |
| 路面衝撃 | 車体が軽く小さい → 段差や凹凸をもろに受ける |
| 半ヘルの風圧・虫 | 顔・首に直接風と虫が当たり続ける → 首・肩が疲れる |

**125ccで改善されること（7〜8割解消）：**
| 項目 | 原付 | 125cc |
|---|---|---|
| 速度制限 | 30km/h | なし（流れに乗れる） |
| エンジン振動 | 常に高回転 | 低回転で余裕 |
| 精神ストレス | 高い | 低い |
| 虫・風 | ヘルメット次第（共通） | ヘルメット次第（共通） |

**残る疲れ：** 車よりは疲れる。長距離・長時間には向かない。
