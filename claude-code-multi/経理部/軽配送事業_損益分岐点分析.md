## ■ 損益分岐点

### 月間コスト総計
```
固定費：         32,500円
燃料（91h）：    22,750円
修理月割り：      6,667円
━━━━━━━━━━━━━━━━━━
合計：          61,917円（約62,000円）
```

### 損益分岐ライン

純利益ゼロになる条件：

**① 時給ベース**
```
固定費+修理 = 39,167円
変動費（燃料）は時給から引く必要がある

損益分岐時給 = 62,000円 ÷ 91h ≈ 681円/h

つまり燃料込みで 681円/h 以上稼げないと赤字
燃料抜きの「純稼ぎ」: 681 + 250（燃料）= 931円/h 必要
```

**② 稼働時間ベース（時給別）**
| 時給 | 損益分岐時間/月 | 最大91hのとき |
|---|---|---|
| 1,167円/h（最悪の実績） | **68h** | 利益 +44,000円 |
| 1,500円/h（並の日） | **50h** | 利益 +74,500円 |
| 1,800円/h（良い日） | **41h** | 利益 +101,800円 |
| 2,500円/h（雨の日ピーク） | **29h** | 利益 +166,000円 |

> 最悪の日（時給1,167円）でも91h全部稼働すれば月+44,000円の黒字。
> ただし**68h稼働が確保できないと赤字**になる。

### 現在の問題点
- 最悪の日（3h = 3,500円 = 時給1,167円）では**ギリギリ**ラインに近い
- 68h必要 / 91h稼働可能 → **余裕は23h分だけ**
- 体調不良や雨回避が続くと赤字転落のリスクあり

## ■ 車両の今後：修理すべきか

### 車の現状評価
```
車両：ハイゼットカーゴ S321V
初年度：2013年5月（現在13歳）
残ローン：約15ヶ月 × 14,500円 ≈ 217,500円
現在の市場価値（現状渡し）：推定 30,000〜80,000円
```

**結論：ローンがある以上、今は修理するしかない。**

理由：
- 売却しても30〜80万円にしかならず、残債217,500円を手出しで払う羽目になる
- オルタネーターを放置すると走れなくなる＝収入ゼロになる
- 2027年7月以降ローンが消えると月14,500円が浮く → その時点で乗り換え検討が現実的

### 修理後の判断ライン
```
オルタネーター + タイヤ = 約75,000円の即時支出が必要
→ これは「車を動かし続けるための必要経費」として割り切る
→ 修理しないと事業継続不可
```

### これからも修理は増えるか？

**ほぼ確実に増える。**

13年落ちの軽貨物で今後想定される追加故障：
| 想定箇所 | 推定費用 | 時期感 |
|---|---|---|
| ブレーキパッド/ローター | 20,000〜40,000円 | 1〜2年以内 |
| ウォーターポンプ・サーモスタット | 20,000〜50,000円 | 1〜3年 |
| CVTオイル/ミッション | 30,000〜80,000円 | 運次第 |
| 車検（2年ごと） | 70,000〜100,000円 | 定期 |

→ **年間100,000〜150,000円の修理予算**は現実的に見ておく必要がある  
→ 月換算：8,333〜12,500円の追加コスト

### コスト比較（月間）

| **合計** | **61,917円** | **約11,000〜16,000円** |

**月間コスト削減額：約46,000〜51,000円**

### 125ccへの初期投資
```
免許取得（AT小型限定）：約100,000円（2〜3日）
中古125cc：80,000〜150,000円
デリバリーボックス・ステー：20,000〜40,000円
ライダーギア（雨具含む）：30,000〜50,000円
━━━━━━━━━━━━━━━━━━
初期投資合計：230,000〜340,000円
```

### 125ccの決定的なデメリット

**雨の日に乗れない = 雨の日の高収入を捨てることになる**

> 雨の日はデリバリー需要が急増し、時給2,000〜2,500円に上がることが多い。
> これを毎回捨てるのはコスト削減額を一部相殺する。

### 乗り換えの結論

**今は乗り換えるタイミングではない。2027年7月以降が現実的。**

理由：
1. 残債217,500円がある → 今売っても損失が出る
2. 免許+バイク購入で300,000円の追加出費 → 資金的に苦しい
3. 彼女（明石）への移動手段を残したいなら車が必要なケースもある

**2027年8月以降の判断基準：**
```
軽貨物を続けるかバイクに乗り換えるかは、
その時点での修理費の多さと月収実績で判断する。

目安：月2回以上の高額修理が出るなら乗り換えサイン
```

## ■ 判断のための数字まとめ

### やるべき行動（今すぐ）
1. **オルタネーター修理：必須**（放置 = 事業停止）
2. **タイヤ4本交換：必須**（雨の日のタイヤ滑り = 事故リスク）
3. **窓ボタン：後回し可**

### 損益分岐の現実
```
月62,000円のコストをペイするには
→ 時給1,167円（最悪ライン）なら 月68時間以上稼働が必要
→ 稼働可能な91hの75%はこなす必要がある

体調不良・雨回避・メンタル低下などで稼働率が下がると赤字転落
```

### 黒字を安定させる条件
- **時給1,500円以上** を安定して出せるエリア・時間帯を固定する
- 稼げる時間帯（ランチ11〜13時、ディナー17〜20時）に集中稼働する
- 雨の日は積極的に出る（最も効率がいい）

### 最悪ケース（事業継続リスク）
```
・月の稼働が50h以下かつ時給1,200円程度なら月収60,000円以下
→ コストとほぼ同額 → 手元に残るお金がほぼゼロ

・修理が重なって月に追加で30,000〜50,000円出費が発生すると赤字確定
→ 借金300万円返済中の現状ではキャッシュが一気に危うくなる
```
