- いつでも解約できる（事業初期フェーズに最適）
- 守口→明石（彼女の家）の移動コストも削減
  - 電車：約2,200〜2,400円/往復
  - 125cc（一般道）：約560円/往復（月2〜3回で3,000〜5,400円節約）

### 移行の注意点
- 雨の日はバイク稼働しない（ただし元々稼働していないため問題なし）
- 高速道路不可（守口〜明石は一般道、約1.5〜2時間）
- 虫問題：**ヘルメット次第。** ジェットヘル+バイザーで解消。半ヘルのままだと解決しない。

## ■ 疲労比較：原付 vs 125cc

現在、原付でフードデリバリーをお試し中。「めちゃくちゃ疲れた」の原因と改善見込み。

**原付が疲れる主な理由：**
| 原因 | 内容 |
|---|---|
| 30km/h制限のストレス | 後続車に常に追われる精神的消耗 |
| エンジン高回転 | 30km/h維持でもエンジンが唸る → 振動が蓄積 |
| 路面衝撃 | 車体が軽く小さい → 段差や凹凸をもろに受ける |
| 半ヘルの風圧・虫 | 顔・首に直接風と虫が当たり続ける → 首・肩が疲れる |

**125ccで改善されること（7〜8割解消）：**
| 項目 | 原付 | 125cc |
|---|---|---|
| 速度制限 | 30km/h | なし（流れに乗れる） |
| エンジン振動 | 常に高回転 | 低回転で余裕 |
| 精神ストレス | 高い | 低い |
| 虫・風 | ヘルメット次第（共通） | ヘルメット次第（共通） |

**残る疲れ：** 車よりは疲れる。長距離・長時間には向かない。
