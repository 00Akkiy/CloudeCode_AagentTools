## ■ 車両の経緯と廃車決定（2026-04-09 引き渡し）

### 車両の詳細
```
車両：ハイゼットカーゴ S321V（平成27年5月・2015年式）
取得経緯：友人がローンで購入 → あつきが譲受・名義変更（黒ナンバー登録）
ローン名義：友人（2027年7月まで残）
あつきの支払い：友人へ14,500円/月（2026年7月まで）
車の所有者：あつき名義（廃車手続きは本人で完結可能）
```

### 廃車の経緯
- オルタネーター故障（充電不可・走行不能）が発覚
- 修理見積もり：約50,000円
- 現状渡し買取額：42,000円（内訳は下記）
- 2026年4月9日：レッカー（保険付帯・無料）で買取業者に引き渡し
- 振込予定：2026年4月15日

**買取交渉の詳細：**
```
当初提示額：      12,000円
「他社比較してない」の一言で交渉：→ 30,000円（+18,000円）
自賠責保険還付金：              +12,000円
最終受取額：                    42,000円

→ 交渉一言で18,000円上乗せ。廃車でも比較交渉は有効。
```

### 廃車の判断が正しかった理由
```
【修理して7月まで乗った場合のコスト試算】
修理代：              50,000円
任意保険 × 4ヶ月：    40,000円
駐車場 × 4ヶ月：      32,000円
合計（ローン除く）：  122,000円の追加支出

【廃車にした場合】
売却収入：           +42,000円
保険・駐車場：        即ゼロ
追加修理リスク：       なし

→ 廃車の方が約122,000〜164,000円 有利な結果
```
