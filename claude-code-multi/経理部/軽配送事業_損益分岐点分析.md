## ■ 損益分岐点

### 月間コスト総計
```
固定費：         32,500円
燃料（91h）：    22,750円
修理月割り：      6,667円
━━━━━━━━━━━━━━━━━━
合計：          61,917円（約62,000円）
```

### 損益分岐ライン

純利益ゼロになる条件：

**① 時給ベース**
```
固定費+修理 = 39,167円
変動費（燃料）は時給から引く必要がある

損益分岐時給 = 62,000円 ÷ 91h ≈ 681円/h

つまり燃料込みで 681円/h 以上稼げないと赤字
燃料抜きの「純稼ぎ」: 681 + 250（燃料）= 931円/h 必要
```

**② 稼働時間ベース（時給別）**
| 時給 | 損益分岐時間/月 | 最大91hのとき |
|---|---|---|
| 1,167円/h（最悪の実績） | **68h** | 利益 +44,000円 |
| 1,500円/h（並の日） | **50h** | 利益 +74,500円 |
| 1,800円/h（良い日） | **41h** | 利益 +101,800円 |
| 2,500円/h（雨の日ピーク） | **29h** | 利益 +166,000円 |

> 最悪の日（時給1,167円）でも91h全部稼働すれば月+44,000円の黒字。
> ただし**68h稼働が確保できないと赤字**になる。

### 現在の問題点
- 最悪の日（3h = 3,500円 = 時給1,167円）では**ギリギリ**ラインに近い
- 68h必要 / 91h稼働可能 → **余裕は23h分だけ**
- 体調不良や雨回避が続くと赤字転落のリスクあり

**【2026-04-08 状況変化】**

**A案：修理して乗り続ける**
- 即時出費：オルタネーター50,000円 + タイヤ25,000円 = **75,000円**

**B案：車を手放して125ccに乗り換え**

- 残債との差額：217,500 − 65,000（平均） = **手出し約152,500円**

**C案：オルタネーター修理せず売却 → 125ccへ**
- 修理費50,000円を節約できる
- ただし故障中の売却価格は下がる（B案とほぼ同額になる可能性）

### 125ccへの初期投資

### 125ccの決定的なデメリット

**雨の日に乗れない = 雨の日の高収入を捨てることになる**

> 雨の日はデリバリー需要が急増し、時給2,000〜2,500円に上がることが多い。
> これを毎回捨てるのはコスト削減額を一部相殺する。

### 原付 vs 125cc：疲労の違い（2026-04-08追加）

現在、原付でフードデリバリーをお試し中。「めちゃくちゃ疲れた」の原因と、125ccでの改善見込み。

**原付が疲れる主な理由：**
| 原因 | 詳細 |
|---|---|
| 30km/h制限 | 後続車に常に追われる精神的消耗。交差点ごとのストレス |
| 高回転エンジン | 30km/h維持でもエンジンが唸る → 振動が手・腰に蓄積 |
| 路面衝撃 | 車体が軽く小さい → 段差や粗い路面をもろに受ける |
| 配達効率の悪さ | 遅いから1件あたりの時間が長い → 疲れた割に稼げない |

**125ccなら7〜8割の疲れは解消される：**
| 項目 | 原付 | 125cc |
|---|---|---|
| 速度制限 | 30km/h | なし（流れに乗れる） |
| エンジン回転 | 常に高回転 | 低回転で余裕がある |
| 精神ストレス | 高い | 低い |
| 路面衝撃 | 大きい | やや改善 |
| 雨の日 | 怖い | やや改善（でも怖い） |

**残る2〜3割の疲れ：** 風・姿勢・雨・信号停止の繰り返しは車よりは疲れる。ただし原付とは別物のレベル。

### 乗り換えの結論（更新）

**オルタネーター故障をきっかけに、125cc移行を本格検討するフェーズに入った。**

移行コスト：470,000〜650,000円（手出し含む）は重い。ただし月間コストは大幅削減。

| 月間コスト | 現在（軽貨物） | 125cc移行後 |
|---|---|---|
| 固定費 | 32,500円 | 5,000〜7,000円 |
| 燃料（91h） | 22,750円 | 約5,700円 |
| 修理月割 | 6,667円〜 | 2,000〜3,000円 |
| **合計** | **約62,000円** | **約13,000〜16,000円** |

**月間削減額：約46,000〜49,000円**
**移行コスト回収期間：約10〜14ヶ月**

→ 長期で見ると125ccのほうが圧倒的にコストが低い。

## ■ 判断のための数字まとめ

### やるべき行動（2026-04-08更新）

**車を手放す方向で検討中のため、判断フローを整理：**

```
① 原付での配達を続けながら「125ccとの疲労差」を体感する
   ↓
② 普通二輪免許を取るかどうか決める（取得コスト約120,000〜150,000円）
   ↓
③ 車はオルタネーター修理せず、現状渡しで売却を打診
   （修理代50,000円を使わず売却損を抑える方が得な可能性）
   ↓
④ 125cc + 配達ボックスで本格稼働
```

**オルタネーターを修理すべきか否か：**
- 修理して売る → 売値UP見込み（+30,000〜50,000円）でも修理代50,000円かかる → ほぼトントン
- 現状渡しで売る → 修理代節約、売値は下がるが実質差額は少ない
- **→ 現状渡し売却が合理的**

### 損益分岐の現実
```
月62,000円のコストをペイするには
→ 時給1,167円（最悪ライン）なら 月68時間以上稼働が必要
→ 稼働可能な91hの75%はこなす必要がある

体調不良・雨回避・メンタル低下などで稼働率が下がると赤字転落
```

### 黒字を安定させる条件
- **時給1,500円以上** を安定して出せるエリア・時間帯を固定する
- 稼げる時間帯（ランチ11〜13時、ディナー17〜20時）に集中稼働する
- 雨の日は積極的に出る（最も効率がいい）

### 最悪ケース（事業継続リスク）
```
・月の稼働が50h以下かつ時給1,200円程度なら月収60,000円以下
→ コストとほぼ同額 → 手元に残るお金がほぼゼロ

・修理が重なって月に追加で30,000〜50,000円出費が発生すると赤字確定
→ 借金300万円返済中の現状ではキャッシュが一気に危うくなる
```
