```
  免許取得：     70,000円
```

### 修理して乗り続けた場合

### 車の収益優位性
- **コストコ配達：時給約4,000円**（20km/1時間・大型荷物）→ バイクでは不可
- **雨の日稼働可能**：需要急増で時給2,000〜2,500円
- 夏冬快適、虫なし → 長時間稼働が楽

### 結論：修理して2027年6月の車検まで乗る

| | 売って125cc | 修理して継続 |
|---|---|---|
| 初期費用 | **141,000円** | **50,000円** |
| 月間コスト（〜7月） | 20,000円 | 32,500円 |
| 月間コスト（8月〜） | 20,000円 | **18,000円** |
| コストコ配達（時給4,000円） | 不可 | **可** |
| 雨の日稼働 | 不可 | **可** |
| 快適性 | 暑い・寒い・虫 | 楽 |

**修理の方が91,000円安く、収益ポテンシャルが高い。**

次の判断ポイント：**2027年6月の車検時**（70,000〜100,000円かかる）にどうするか決める。

## ■ 疲労比較：原付 vs 125cc

現在、原付でフードデリバリーをお試し中。「めちゃくちゃ疲れた」の原因と改善見込み。

**原付が疲れる主な理由：**
| 原因 | 内容 |
|---|---|
| 30km/h制限のストレス | 後続車に常に追われる精神的消耗 |
| エンジン高回転 | 30km/h維持でもエンジンが唸る → 振動が蓄積 |
| 路面衝撃 | 車体が軽く小さい → 段差や凹凸をもろに受ける |
| 半ヘルの風圧・虫 | 顔・首に直接風と虫が当たり続ける → 首・肩が疲れる |

**125ccで改善されること（7〜8割解消）：**
| 項目 | 原付 | 125cc |
|---|---|---|
| 速度制限 | 30km/h | なし（流れに乗れる） |
| エンジン振動 | 常に高回転 | 低回転で余裕 |
| 精神ストレス | 高い | 低い |
| 虫・風 | ヘルメット次第（共通） | ヘルメット次第（共通） |

**残る疲れ：** 車よりは疲れる。長距離・長時間には向かない。
